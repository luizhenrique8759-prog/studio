# **App Name**: Clínica Dental Sync

## Core Features:

- Agendamento de Consultas Online: Permite aos usuários selecionar serviços, datas e horários disponíveis para agendar suas consultas odontológicas de forma conveniente.
- Exibição de Disponibilidade Dinâmica: Mostra aos usuários em tempo real quais horários estão livres ou ocupados para facilitar a escolha de agendamento.
- Painel Administrativo para Gestão: Uma interface exclusiva para o administrador visualizar, gerenciar e confirmar todos os agendamentos.
- Gestão de Usuários e Profissionais: Recursos administrativos para criar, editar ou desativar contas de pacientes e dentistas na plataforma.
- Confirmação e Notificações de Agendamento: Administradores podem confirmar agendamentos e o sistema pode enviar notificações automáticas aos usuários.
- Ferramenta IA para Resumos de Faturamento: Uma ferramenta de IA que gera resumos detalhados dos serviços e consultas para auxiliar no processo de faturamento.
- Persistência de Dados em Firestore: Todos os dados de usuários, agendamentos e informações da clínica serão armazenados de forma segura e escalável no Firestore.

## Style Guidelines:

- Primary color: `#2E93CC` (a fresh, clean blue reflecting professionalism and trustworthiness). Its HSL value is around (200, 60%, 45%).
- Background color: `#ECF2F5` (a very light, desaturated blue derived from the primary, promoting a clean and bright interface). Its HSL value is around (200, 20%, 95%).
- Accent color: `#53DBBD` (an aquamarine shade, analogous to the primary but providing a distinct visual highlight for interactive elements and calls to action). Its HSL value is around (170, 70%, 55%).
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-oriented and clear aesthetic. Body text font: 'Inter' (sans-serif) for high readability and a neutral, professional appearance, complementing 'Space Grotesk'.
- Utilize a consistent set of clean, line-based icons for navigation, scheduling, user profiles, and administrative functions, conveying professionalism and clarity.
- Employ a spacious, grid-based layout with generous use of whitespace to enhance readability and reduce cognitive load for users and administrators. Ensure responsiveness across devices.
- Implement subtle and smooth transitions for calendar navigation, appointment selections, and form submissions to provide a fluid and responsive user experience without distraction.
